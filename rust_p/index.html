Rust 语言入门教程

教程概述

这是一个专为初学者设计的Rust语言教程，旨在帮助您在一天内掌握Rust的核心概念并通过实践项目熟练掌握Rust API。

学习路径

一：基础概念 (2-3小时)

  1. 环境搭建和Hello World
  2. 变量和数据类型
  3. 控制流
  4. 函数

二：核心特性 (3-4小时)

  1. 所有权系统
  2. 借用和引用
  3. 结构体和枚举
  4. 集合类型

三：高级特性 (2-3小时)

  1. 错误处理
  2. 模块和包管理
  3. 标准库学习指南
  4. 泛型和trait

四：实践项目 (2-3小时)

  1. 命令行计算器
  2. 文件处理工具
  3. 简单Web服务器

五：进阶主题

  1. 练习题集

六：其他参考资料

宏系统

  1. 过程宏详解
  2. 派生宏详解
  3. 声明宏详解

其他语法

  1. Attribute语法
  2. 模块系统详解（mod.rs vs Python）
  3. mod.rs和lib.rs的区别
  4. 可见性修饰符对比
  5. OnceLock 和 OnceCell

七：UniFFI - 跨语言调用

  1. UniFFI 导出详解

八：Tokio 异步编程

  1. Tokio 入门
  2. 异步编程基础
  3. Tokio Runtime
  4. Tokio vs 其他运行时
  5. 实战项目

快速开始

# 安装Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 验证安装
rustc --version
cargo --version

# 运行第一个程序
cargo new hello_world
cd hello_world
cargo run

学习建议

  1. 按顺序学习：每个章节都建立在前一章的基础上
  2. 动手实践：每个概念都配有相应的代码示例
  3. 完成练习：每章末尾都有练习题，务必完成
  4. 运行代码：不要只看代码，一定要运行并修改
  5. 查阅文档：遇到问题时查阅官方文档
